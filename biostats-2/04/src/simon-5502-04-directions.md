## Question 1

Create a child variable using the code you saw in class. Also create an improvement variable that calculates how much better the results in trial4 are compared to trial1. Calculate and interpret a histogram for the improvement variable.

## Question 2

Draw a line plot that examines whether there might be an interaction between child and gender in predicting improvement. You should look at the plot two different ways, but only include the one plot that you think best describes what is going on. Interpret this plot.
